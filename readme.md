# Horos - Astrology App

## Prerequisites

- Node.js (v18+)

## Setup

```bash
# Install frontend dependencies
npm install

# Install server dependencies
cd server
npm install
cd ..
```

## Running the App

```bash
# Start the backend server (in one terminal)
cd server
npm start

# Start the frontend dev server (in another terminal)
npm run dev
```

## Build for Production

```bash
npm run build
npm run preview
```
